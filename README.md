# Gym Membership management dashboard with Flask and MongoDB

A full-stack **NoSQL web application** built with **Python (Flask)** and **MongoDB** for managing gym customers, memberships, and visit logs.

---

## Features

### Customer Management (CRUD)
- Add, edit, delete, and search customers  
- Store personal details (name, email, phone)  
- Link customers to memberships  

### Membership Management (CRUD)
- Create and manage memberships  
- Pricing categories: Student, Normal, Pensioner, Veteran  
- Track active membership and valid dates  

### Visit Log Tracking
- View entry and exit logs per membership  
- Monitor gym usage history  

###  Dynamic UI
- Switch between customer and membership views  
- Interactive forms with pre-filled edit data  
- Clean UI with vanilla JavaScript  

---

## Tech Stack

- **Backend:** Python, Flask  
- **Database:** MongoDB (PyMongo)  
- **Frontend:** HTML, CSS, JavaScript  

---


## How to Run

```bash
git clone https://github.com/poddui/GymMembershipMongoDB.git
cd GymMembershipMongoDB
pip install flask pymongo
python app.py
`````
